## 🚀 Características

- **Calculadora de Integrales Definidas**: Resuelve integrales con límites de integración.
- **Integrales Indefinidas**: Calcula antiderivadas de funciones polinómicas.
- **Gráficas Interactivas**: Visualiza el área bajo la curva con Matplotlib.
- **Interfaz Moderna**: Tema oscuro con CustomTkinter.

## ⚙️ Requisitos

- Python 3.11 o superior
- `customtkinter`
- `matplotlib`
- `sympy`

## 🛠️ Instalación

1. **Clona el repositorio** (o descarga el código fuente)

   ```bash
   git clone https://github.com/yourusername/Integrator-App.git
   cd Integrator-App
   ```

2. **Instala las dependencias**

   ```bash
   pip install customtkinter matplotlib sympy
   ```

## 🏃‍♂️ Ejecución

Simplemente ejecuta el script principal:

```bash
python main.py
```

## 📋 Uso

1. Selecciona el tipo de integral (Definida o Indefinida).
2. Introduce la función `f(x)` usando notación matemática (ej. `x**2 + 3*x + 5`).
3. Para integrales definidas, ingresa los límites `a` y `b`.
4. Presiona "Calcular".
5. Opcionalmente, presiona "Gráfica" para ver el área bajo la curva.
